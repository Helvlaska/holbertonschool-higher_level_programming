Test 1 - Addition simple :
a = 1, b = 2 → Résultat attendu : 3

Test 2 - Un seul argument fourni :
a = 10 → Résultat attendu : 108

Test 3 - Float convertis en int :
a = 3.5, b = 2.7 → Résultat attendu : 5 (int(3.5) + int(2.7))

a = -1.9, b = -3.1 → Résultat attendu : -4 (int(-1.9) + int(-3.1))

a = "hello", b = 3 → Erreur attendue : TypeError("a must be an integer")

a = 3, b = "bonjour" → Erreur attendue : TypeError("b must be an integer")

Test 6 - a est None :
a = None, b = 3 → Erreur attendue : TypeError("a must be an integer")

a = 3, b = None → Erreur attendue : TypeError("b must be an integer")
